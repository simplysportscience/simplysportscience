Information Processing

What actually happens between seeing something on the pitch and reacting to it — and why a well-timed dummy genuinely buys a performer extra time.

Key Terms

Information Processing ModelDisplay → sensory information → perception → decision making → effector mechanism → response → feedback.
Knowledge of Results (KR)Feedback about the outcome of a performance.
Knowledge of Performance (KP)Feedback about the quality of technique, regardless of outcome.
Selective AttentionFiltering relevant sensory information from irrelevant information.
Psychological Refractory PeriodA delay in responding to a second stimulus presented shortly after a first.

7.1 Theory of Information Processing

The information processing model describes how a performer moves from sensing to acting: display (the environment) → sensory information received via sense organs → perception → decision making → effector mechanism (muscles) → response → feedback.

7.2 Feedback

Feedback is essential to learning and performance improvement. Types you must know:

  • Intrinsic (from within the performer) vs extrinsic (from an external source).
  • Terminal (given after a performance) vs concurrent (given during a performance).
  • Positive vs negative.
  • Knowledge of performance (KP) — feedback on technique quality, vs knowledge of results (KR) — feedback on the outcome.

7.3 Memory

The memory process: short-term sensory store (briefly holds all incoming sensory information) → selective attention (filters relevant from irrelevant information) → short-term memory (limited capacity, temporary) → long-term memory (relatively permanent storage). Strategies to improve storage in long-term memory include chaining information together, using mental rehearsal, and frequent, distributed practice.

7.4 Response Time

Response time = reaction time + movement time. Reaction time is the time from stimulus to the start of a response; movement time is the time to complete the movement once started.

Factors affecting reaction time include:

  • Simple vs choice reaction time — choice reaction time is slower, since more than one possible response must be considered.
  • Hick's Law — reaction time increases as the number of possible response choices increases.
  • Psychological refractory period (PRP) — a delay in responding to a second stimulus presented shortly after a first, since the system can only process one stimulus at a time.
  • Single-channel hypothesis — the underlying explanation for the PRP.

These concepts explain strategies to deceive opponents (e.g. a dummy pass exploits the PRP) and strategies to improve response time (e.g. reducing the number of realistic response choices through anticipation).

Exam tip: "Explain why a dummy or feint is effective" is a PRP/single-channel hypothesis question — name the concept directly, not just "it confuses the opponent."

Practice Check
1. Describe the stages of the information processing model. +
Model answer points:
  • Display — the environment or situation.
  • Sensory information — received via the sense organs.
  • Perception and decision making — interpreting the information and selecting a response.
  • Effector mechanism — the muscles carrying out the response.
  • Feedback — information about the outcome, used to inform future responses.
2. Explain the difference between knowledge of results and knowledge of performance. +
Model answer points:
  • Knowledge of results is feedback about the outcome of a performance, e.g. whether a shot scored.
  • Knowledge of performance is feedback about the quality of technique used, regardless of the outcome.
3. Explain the role of selective attention in the memory process. +
Model answer points:
  • Sensory information first enters the short-term sensory store, which briefly holds all incoming information.
  • Selective attention filters relevant information from irrelevant information.
  • Only the selected, relevant information passes into short-term memory for further processing.
4. Explain, using the psychological refractory period, why a dummy movement can be effective in sport. +
Model answer points:
  • The single-channel hypothesis proposes that the information processing system can only fully process one stimulus at a time.
  • A dummy movement presents a first stimulus that the opponent begins to process and respond to.
  • This delays the opponent's ability to process the second, genuine stimulus that follows — the psychological refractory period — giving the performer a timing advantage.
